# CodeAlpha_Portfolio_Tracker
This repository contains a financial tracking tool developed for the CodeAlpha Python Programming Internship. 

## Features
* Calculates total investment values based on hardcoded stock prices (AAPL, TSLA, GOOGL, etc.).
* Utilizes Python dictionaries to store, retrieve, and map user inputs to corresponding data points.
* Handles user inputs cleanly, automatically formatting text to prevent case-sensitive errors.
* Includes file handling capabilities (`with open`) to automatically export a summary of the user's portfolio to a `.txt` file.
